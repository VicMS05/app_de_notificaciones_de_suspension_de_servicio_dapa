1.- Instalar la carpeta drf que contendrá las herramientas de Django con el comando "django-admin startproject drf .".
2.- Crear la API usando el comando "django-admin startapp api".
3.- En drf/settings.py agregar 'api' al arreglo INSTALLED_APPS.
4.- En la misma carpeta configurar el arreglo DATABASES para que tenga variables para una conexión a una base de datos de MySQL, ya que por defecto viene para una conexión para SQLite3.
